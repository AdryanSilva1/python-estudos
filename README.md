# 🐍 Curso em Vídeo — Python 3

Este repositório reúne os exercícios que resolvi durante o **Curso em Vídeo de Python**, ministrado pelo professor **Gustavo Guanabara**.

O objetivo deste projeto é registrar meu processo de aprendizado em Python por meio da prática, documentando tentativas, correções e melhorias de código ao longo do curso.

Cada exercício representa uma etapa da evolução na linguagem — desde os primeiros conceitos até estruturas de dados e organização de código.

---

## 📂 Organização dos exercícios

Cada arquivo segue uma estrutura pensada para acompanhar a evolução do aprendizado:

- **Minha solução** — implementação inicial escrita durante o estudo, incluindo erros comentados.
- **Solução corrigida** — versão funcionando corretamente após revisão.

Essa abordagem permite visualizar claramente o processo de aprendizado e aprimoramento do código.

---

## 📚 Estrutura do curso

O curso é dividido em três módulos principais, chamados de *Mundos*:

### 🌍 Mundo 1 — Fundamentos
** Exercícios** abordando os conceitos iniciais da linguagem:

- entrada e saída de dados  
- operadores aritméticos  
- manipulação de strings  
- condicionais básicas  
- primeiros laços de repetição  

Essa etapa estabelece a base da lógica de programação em Python.

---

### 🌍 Mundo 2 — Estruturas de Controle
** Exercícios** aprofundando o controle do fluxo do programa:

- estruturas `if`, `elif` e `else`  
- laços `while` e `for`  
- validação de dados  
- primeiros contatos com funções  

Aqui a lógica começa a se estruturar de forma mais consistente.

---

### 🌍 Mundo 3 — Estruturas de Dados
** Exercícios** explorando estruturas mais complexas da linguagem:

- listas  
- tuplas  
- dicionários  
- matrizes  
- funções com parâmetros e `*args`  

Inclui também um pequeno projeto modular com organização em pacotes e uso de `__init__.py`.
